# Queueing-System
During our OJT we provided a Queueing System for Company to be used as good.
Please Do not re-upload the System, This is for educational purpose only.
System is good for localhosting using XAMMP, there is a 2 main users: Admin and the Cashier, admin has the CRUD operations for each cashier registered.
in the login system, there is also an option to create user Admin or Cashier.
We have not used database to store each Cashier or admin accounts. We only used .json to store each data, and as well as for the Queues.
system also features Automatic calling of number via Browser.
